    Note：

    著者は小学館の編集者で、「週刊少年サンデー」の後、「裏サンデー」「マンガワン」を達あえげた人。
    プロの漫画家になる上でのTipsを、企画からキャラクター造型、ストーリー構成から、コマ割やフキダシや集中線の効果的な使い方など具体的な技法まで、取り上げている内容は幅広い。編集者とのトラブル回避法なんて節もあるのが現代的だなと。

    現役編集者が描いた漫画技法書というと最近だと大場渉の「マンガの原理」に比べると、こちらはやや抽象的にアドバイスが多いかなという印象。

    基本的にはプロの漫画家を目指す人向けの本だが、単に漫画を読むだけの人にとっても、漫画家や編集者がどのように考え、どう意識して漫画を描いているのかを知る手がかりになる。

    個人的に印象的だったのは、ネームの段階で編集者がかなり細かくダメ出しをしているという点で、これは著者の担当する漫画家が若手が多いということもあるんだろうけど、コマの配置や構図など事細かにチェックしており、完全にアドバイスやサジェストの域を超えて監修というべきレベルで、いろいろ考えさせられる。
