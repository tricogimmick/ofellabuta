    暴力都市

    タイトル：暴力都市（原題：살인청부업자／HIT MAN）
    国：韓国
    公開年：2022年
    視聴：2025-08-25 - U-Next
    監督： カン・テホ
    脚本： チョ・ドンヒ
    出演： チョン・ジェウ / パク・ジス / コ・ウォンソク / ソル・ジェグン
    備　考：

    2022年の韓国映画。

    Note：

    息子一家の殺害を依頼された殺し屋が、犯人を数珠つなぎに拷問・殺害しながらラスボスを目指すというノワールもの。
    予算がないのだろうかVシネ感漂う画面。

    基本、スタンガンで失神させれて拉致→トラックのコンテナなの中で指を切断していく拷問屋→ラップで顔をぐるぐる巻きにして殺害というパターンの繰り返しなので展開が単調かつ地味。主人公の殺し屋の背景もとってつけた感じで物語に絡んでこないし、2秒で殺し屋側の素性が犯人側に突き止められてる展開とかいろいろ拙い箇所が目立つ作品。

    でも結構嫌いじゃないかも。主人公の殺し屋のサイドキックを務めてる兄ちゃんが結構良い味だしてる。多分ないだろうけど、続編があるなら観てみたい。

    OSAMU NOMURA / TRICOGIMMICK 2024-2025
