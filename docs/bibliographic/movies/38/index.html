    犯罪都市

    タイトル：犯罪都市（原題：THE OUTLAWS）
    国：韓国
    公開年：2017年
    視聴：2025-08-26 - U-Next
    監督： カン・ユンソン
    脚本： カン・ユンソン
    出演： マ・ドンソク / ユン・ゲサン / チョ・ジェユン / チェ・グィファ / パク・ジファン / チン・ソンギュ / ホ・ソンテ / キム・ソンギュ
    備　考：

    2017年の韓国映画。
    2004年に実際にあった事件をもとにした作品、ヤクザVS警察モノ。
    主演の刑事にマ・ドンソク。

    Note：

    何故か銃器が出てこない。ヤクザは刃物、警察の方はステゴロというスタイルは、マ・ドンソクのフィジカルな魅力を最大限に引き出すためのものだろうか。マ・ドンソクの強面だが愛嬌のあるキャラクターはやはり魅力的。

    悪役の朝鮮族中国マフィアのチャン・チェンの極悪非道ぶりはなかなかのものだが「孤狼の血 LEVEL2」の鈴木亮平演じる上林ほどの狂気は感じられず。暴力的ではあるが韓国ノワール的な凄惨かつダークな要素は控えめで娯楽作に徹してるのは正解。

    ただラスト、食堂のおっさんと少年のその後が気になる。フォローがあっても良かったのでは。

    OSAMU NOMURA / TRICOGIMMICK 2024-2026
