    妖怪ハンター 天の巻

    タイトル：妖怪ハンター 天の巻
    出版種別：書籍
    作者：諸星大二郎
    出版社：集英社（集英社文庫）
    発行日：2005-11-18
    購入日：2025-10-17
    読了日：2025-10-19
    管理番号：
    Note：

    集英社版「妖怪ハンター」集大成の２巻は、おもに天木兄妹と橘が登場する「生命の種」を題材とする連作を中心にまとめた内容。

    このシリーズ、稗田は知恵も知識もあり勇敢だけど、基本的に超常現象に対しては無力なのが良い。
