    人間爆弾 立ち止まったら、爆発

    タイトル：人間爆弾 立ち止まったら、爆発（原題：TODOS LOS NOMBRES DE DIOS）
    国：スペイン
    公開年：2023年
    視聴：2025-04-04 - U-Next
    監督：ダニエル・カルパルソロ
    脚本：ジェマ・ヴェンチュラ
    出演：ルイス・トサル / インマ・クエスタ / ロベルト・エンリケス / パトリシア・ビコ
    リンク：allcinema
    備　考：

    2023年のスペイン映画。
    イスラム過激派の自爆テロ現場に居合わせたタクシー運転手が死に損ねたテロリストの青年を助けようとして逆に人質として誘拐され、失敗したテロリストの代わりに人間爆弾にされてしまうというサスペンス。

    Note：

    主人公がテロリストに人間爆弾にされてしまうのが映画の中盤からで、それまでの前振りはかなり冗長に感じる。前半を費やす死に損ねたテロリストの青年と主人公の交流が後半の展開にほとんど関係ないというのがそう感じる要因だろう。

    また、歩き続けなければ爆発するという仕掛けも、その部分に関するサスペンスが何もないのが勿体ない。面白くなりそうなアイデアなだけに脚本や演出の作り込みの甘さが残念。

    OSAMU NOMURA / TRICOGIMMICK 2024-2026
