    インビジブル・ゲスト 悪魔の証明

    タイトル：インビジブル・ゲスト 悪魔の証明（原題：CONTRATIEMPO）
    国：スペイン
    公開年：2016年
    視聴：2025-02-17 - U-Next
    監督： オリオル・パウロ
    脚本： オリオル・パウロ
    出演： マリオ・カサス / アナ・ワヘネル / ホセ・コロナド / バルバラ・レニー / フランセスク・オレーリャ
    リンク：allcinema
    備　考：

    2016年のスペイン映画。
    不倫相手の殺害容疑で起訴された実業家の主人公。現場は完全な密室で圧倒的に主人公に不利な状況。そこに表れる凄腕女性弁護士。審理が開始されるまでの3時間に間に状況を覆そうと弁護士による主人公への聞き取りが開始される。というミステリ作品。

    監督・脚本はミステリ＋オカルトという異色作「ロスト・ボディ」のオリオル・パウロ。

    Note：

    映画は全編、主人公と弁護士の会話からの回想となっているが、徐々に主人公は信頼できない語り手であることが明らかになっていく。
    巧に張られた伏線と仕組まれた違和感がラストの意外な結末に収斂していくさまは見事。

    さらに最後まで見た後に、映画の冒頭を見直すことでさらに驚きがあるという仕掛けは凄い。

    OSAMU NOMURA / TRICOGIMMICK 2024-2025
