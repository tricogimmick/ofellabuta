おい、君、江戸川乱歩というのは平井太郎だぜ。

「キビキビした青年紳士」甲賀三郎 より

甲賀三郎の随筆の一節。初出は昭和2年の「大衆文藝」。

甲賀三郎が友人から、新進の探偵小説作家江戸川乱歩の正体がかつて親交のあったとある倶楽部の書記長だった青年紳士であったと知らされたさいの友人の台詞。

一度は言ってみたい台詞だ。
