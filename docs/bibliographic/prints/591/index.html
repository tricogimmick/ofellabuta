    監修：東雅夫
    出版社：山と渓谷社
    発行日：2016-07-21
    購入日：2025-12-08
    読了日：2025-12-08
    管理番号：
    Note：

    子供向けでも、専門家やマニア向けでもない、ごく普通の大人向けの妖怪図鑑。
    図鑑とはいいつつ、読み物としてもなかなかの内容。妖怪好きにはやや物足りない感じではあるけれど、現在における「妖怪」について幅広くコンパクトにまとめている。

    アマヤギ堂の妖怪画についても水木しげるとは違うテイストで、なかなか可愛らしくて良い。
